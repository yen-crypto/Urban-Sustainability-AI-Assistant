# Urban-Sustainability-AI-Assistant
「城市永續觀察與基地分析 AI 助手」iOS App，協助使用者快速理解城市空間與環境問題。現有的 GIS、ArcGIS 或都市分析工具雖然功能強大，但多半需要專業背景與較高操作門檻；因此我希望做的是一個更輕量、直覺，適合學生、居民、遊客或規劃初學者使用的城市觀察工具。  基本操作邏輯是，使用者打開 App 後，可以先選擇「輸入地點」或「上傳／拍攝現場照片」。接著 App 會透過 AI 問答方式，引導使用者回答幾個簡單問題，例如使用者身份是居民、學生、遊客或規劃者，這個地點目前觀察到的主要問題是什麼，以及最希望改善的空間面向。系統會根據使用者輸入的地點、照片與回答內容，自動生成一份簡短的基地觀察摘要，包含問題分類、可能原因與初步改善方向。

# Urban Sustainability AI Assistant

This is an early-stage iOS app concept for helping users understand urban sustainability issues through AI-assisted site observation.

The app allows users to input a location or upload a photo. AI will guide users through simple questions, identify possible urban and environmental issues, and generate a short site observation summary with visual analysis suggestions.

Core features:
- Location-based urban observation
- Photo-based site analysis
- AI-guided questions
- Simplified issue classification
- Visual analysis for walkability, green space, heat island risk, flooding risk, vacant space, and local identity
- Exportable observation notes for campus improvement, community proposals, and urban planning learning

This idea is still in the concept and prototype stage.
